# FastAPI Template

This is a template repository for creating FastAPI applications.

## Features

- FastAPI
- Docker
- PostgreSQL
- GitHub Actions for CI
- Configuration management
- Testing setup

## Getting Started

### Prerequisites

- Docker
- Docker Compose

### Installation

1. Clone the repository:

```sh
git clone https://github.com/your-username/fastapi-template.git
cd fastapi-template
```
